# CityFlow

CityFlow è una piattaforma civica open source per le città.

Il progetto integra:

- eventi cittadini
- attività commerciali
- offerte locali
- segnalazioni urbane
- comunicazioni civiche

Il progetto nasce a Fidenza ed è promosso dal
Comitato Legalità e Decoro per Fidenza.

## Tecnologie

- PostgreSQL
- PostGIS
- PostgREST
- Flutter

## Obiettivo

Creare una piattaforma digitale che migliori la comunicazione tra cittadini, attività locali e comunità.

## Stato progetto

Fase iniziale – prototipo città pilota: Fidenza.
